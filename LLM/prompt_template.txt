  - input_data: Array of input data
  - steps: Array of steps to execute

INPUT:

[USER_STORY]

Critères d'acceptation:
[ACCEPTANCE_CRITERIA]

OUTPUT:
- JSON array of test cases covering the acceptance criteria, with positive, negative and boundary cases if applicable.
